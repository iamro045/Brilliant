# Brilliant.org Clone

A modern, interactive learning platform clone designed to mirror the intuitive and gamified experience of Brilliant.org. This project focuses on providing a clean, user-centric interface for educational content, tracking user progress, and encouraging daily learning habits.



## ✨ Key Features
*   **Interactive Dashboard:** A centralized hub to view active courses and personal progress.
*   **Gamified Experience:** Integrated streak counting and experience point (XP) tracking to motivate daily participation.
*   **Course Navigation:** A structured grid layout for browsing lessons with seamless transitions between modules.
*   **Challenge Screen:** An interactive environment for users to test their knowledge with guided feedback.
*   **User Authentication:** Secure access to personal accounts to save progress and maintain history.

## 🛠️ Tech Stack
*   **Frontend:** React, Vite, CSS (Custom styling for a minimalistic, airy aesthetic)
*   **Backend:** Node.js
*   **Deployment:** GitHub Pages
